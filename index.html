Front-End

Para que serve?

O desenvolvedor front-end é responsável pela
experiência do usuário dentro de uma
aplicação web, é ele quem vai desenhar e
desenvolver as páginas com as quais,
posteriormente, o usuário irá interagir.

Back-End

Para que serve?

back-end é tudo aquilo que está por trás
da interface de uma aplicação: seus
sistemas, banco de dados, toda parte de
segurança de dados, envio e recebimento
de informações, armazenamento e etc.
